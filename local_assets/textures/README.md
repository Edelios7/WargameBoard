# Textures & motifs génériques

Comme les autres dossiers de `local_assets/`, jamais commité (voir
`.gitignore`) sauf ce README.

## Contenu

16 textures thématiques génériques (pas liées à une faction ou une
fiche précise), découpées depuis la planche de référence
`wh40k_reference/PDFS/deco et texture/deco et texture.png` :

```
metal-imperial.png       gold-ceremonial.png     chaos.png            corrupted-decay.png
technological.png        mechanicus-sacred.png   psychic-energy.png   xenos-alien.png
urban-ruins.png          desert.png              glacial.png          jungle-overgrown.png
banners-fabric.png       grates-plates.png       symbols-runes.png    energy-lights.png
```

Chaque fichier est une bande large (plusieurs motifs juxtaposés),
pensée comme fond décoratif (carte, en-tête de section...) plutôt que
comme icône — à utiliser avec un dégradé ou une teinte assombrissante
par-dessus pour garder le texte lisible, comme pour
`local_assets/banners/`.

Pas encore branchées dans l'UI : à assigner selon les sections qui
gagneraient à avoir une ambiance visuelle (ex. `gold-ceremonial` pour
une section "prestige"/XP, `metal-imperial` en fond générique, etc.).
