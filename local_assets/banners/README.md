# Bannières de faction

Comme `local_assets/factions/`, ce dossier n'est **jamais commité**
(voir `.gitignore`).

## Convention

Nomme chaque fichier avec l'`id` exact de la faction, comme pour
`local_assets/factions/` :

```
local_assets/banners/fac-blood-angels.png
local_assets/banners/fac-orks.png
```

Contrairement à `local_assets/factions/` (portrait carré, utilisé en
icône), ce sont des bannières larges (ratio proche de 10:1) pensées pour
habiller l'arrière-plan d'une ligne ou d'une section plutôt que servir
d'icône — utilisées par exemple derrière les lignes de progression par
faction de la page Profil.
